# Seafile Dark Mode Theme
If you want to change the default colors, just change these top values

```css
/* start dark theme */

:root {
    /* Backgrounds */
    --main-bg: #2e2e2e;
    /* Soft dark gray â€” base background */
    --dark-bg: #383838;
    /* Elevated surfaces like cards */
    --light-bg: #424242;
    /* Modals, popups, highlights */

    /* Fonts */
    --main-font: #ffffff;
    /* Clean, light gray text */
    --light-font: #b0b0b0;
    /* Medium emphasis */
    --dark-font: #888888;
    /* Low emphasis */

    /* Theme Colors */
    --main-theme: #5a9bd8;
    /* Soft muted blue (accessible but subtle) */
    --theme-font: #f5f5f5;
    /* White-ish for links and emphasis */

    /* Borders and Effects */
    --border: #3c3c3c;
    /* Subtle border against dark-bg */
    --shadow: rgba(0, 0, 0, 0.15);
    /* Gentle depth */
    --no-shadow: rgba(255, 255, 255, 0.02);

    /* Status Colors */
    --green: #85d0a1;
    /* Softer green */
    --info: #f3b5c9;
    /* Muted pink */
    --error: #ef7676;
    /* Gentle red */

    /* Table Headers */
    --th-color: #a0a0a0;
}
```
